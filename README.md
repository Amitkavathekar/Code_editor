## ✨ Features

- **Live Preview** – The output updates instantly in the iframe when you write HTML, CSS, or JavaScript.
- **Syntax Highlighting** – Ace Editor provides colorful syntax highlighting for HTML, CSS, and JavaScript.
- **Autocomplete** – Shows suggestions and code snippets while typing.
- **Dark / Light Mode** – Users can switch between dark and light themes.
- **Copy & Clear** – Each editor includes Copy and Clear buttons.
- **Export to PDF** – Download all the written code in a single PDF file.
- **About Modal** – Displays information about the project.

---

## 🛠 Tech Stack

- **HTML5** – Structure of the project
- **CSS3** – Styling and layout
- **JavaScript (ES6+)** – Application logic
- **Ace Editor (CDN)** – Code editor, syntax highlighting, and autocomplete
- **jsPDF (CDN)** – Export code to a PDF file
